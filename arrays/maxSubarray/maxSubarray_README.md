# Description
Given an array, find the subarray (contiguous values) that has the largest sum and return the sum

For example, given the array

```
[7, 1, -5, 0, 2, 98, -104, 6, 8, 10, -6, 3]
```

the following subarray has the largest sum

```
[7, 1, -5, 0, 2, 98]
```

and the returned value will be

```
103
```

## Constraints

## Input
An array of integers

## Output
The largest sum of a subarray that can be found within the given array
